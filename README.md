# BlueCollar Structural Steel SU Shapes

Free SketchUp shape pack generated from AISC v16.0 structural steel data.

## Included Families

- 2L
- C
- HP
- HSS RECT
- HSS ROUND
- L
- M
- MC
- MT
- PIPE
- S
- ST
- W
- WT

## Files

This repository includes `.skp` family packs for the above shape groups.
